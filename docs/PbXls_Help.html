概述

PbXls 是一个纯 PureBasic 实现的 Excel 文件操作库，无需安装 Microsoft Office 或任何第三方依赖，即可创建和读取 Excel xlsx/xlsm 文件。

该库参考 Python 的 openpyxl 项目移植而来，使用 PureBasic 内置的 XML 和 Packer 库实现。

注意：使用 PbXls 库前，需要在代码开头添加 XIncludeFile "PbXls.pb" 以引入库文件。库会自动初始化，无需手动调用初始化函数。

主要功能

  • 创建和读取 Excel xlsx/xlsm 文件
  • 读写单元格数据（字符串、数值、公式、布尔、日期等）
  • 多工作表支持
  • 单元格样式（字体、填充、边框、对齐、数字格式）
  • 合并单元格
  • 行列宽高设置
  • 列插入/删除（自动更新单元格、列宽、合并单元格）
  • 数据验证（下拉列表、整数验证、小数验证等）
  • 条件格式（单元格值比较、公式条件、颜色刻度）
  • 图表支持（柱状图、折线图、饼图等）
  • 共享字符串表优化
  • 完整的 ZIP 压缩支持

工作簿操作

PbXls_CreateWorkbook()
创建一个新的 Excel 工作簿。
返回值： 工作簿ID（整数）。失败时返回 -1。
; 创建一个新工作簿 wbId = PbXls_CreateWorkbook() If wbId >= 0 ; 工作簿创建成功 EndIf
PbXls_LoadWorkbook(filename.s)
加载一个现有的 Excel 工作簿文件。
参数类型说明
filenameStringExcel文件的完整路径
返回值： 工作簿ID（整数）。失败时返回 -1。
注意： 读取功能当前为占位实现，完整读取功能将在后续版本中实现。
PbXls_SaveWorkbook(wbId.i, filename.s)
将工作簿保存为 Excel 文件。该函数会生成符合 Office Open XML 标准的 xlsx/xlsm 文件。
参数类型说明
wbIdInteger工作簿ID（由 CreateWorkbook 或 LoadWorkbook 返回）
filenameString输出文件的完整路径（.xlsx 或 .xlsm 后缀）
返回值： 成功返回 #True，失败返回 #False。
; 保存工作簿到文件 If PbXls_SaveWorkbook(wbId, "output.xlsx") Debug "保存成功！" Else Debug "保存失败！" EndIf
PbXls_GetSheetCount(wbId.i)
获取工作簿中的工作表数量。
参数类型说明
wbIdInteger工作簿ID
返回值： 工作表数量（整数）。

工作表操作

PbXls_AddWorksheet(wbId.i, title.s = "")
在工作簿中添加一个新的工作表。
参数类型说明
wbIdInteger工作簿ID
titleString工作表名称。如果为空，将自动生成 "SheetN" 格式的名称
返回值： 工作表ID（整数）。失败时返回 -1。
; 添加一个默认名称的工作表 wsId1 = PbXls_AddWorksheet(wbId) ; 结果: Sheet2 ; 添加一个指定名称的工作表 wsId2 = PbXls_AddWorksheet(wbId, "数据统计") ; 结果: 数据统计
PbXls_DeleteWorksheet(wbId.i, index.i)
删除工作簿中指定索引的工作表。
参数类型说明
wbIdInteger工作簿ID
indexInteger要删除的工作表索引（从0开始）
返回值： 成功返回 #True，失败返回 #False。
注意： 工作簿至少需要保留一个工作表，如果只剩一个工作表则无法删除。
PbXls_GetSheetByIndex(wbId.i, index.i)
根据索引获取工作表。返回工作表指针，用于后续操作。
参数类型说明
wbIdInteger工作簿ID
indexInteger工作表索引（从0开始）
返回值： 工作表指针。如果索引无效则返回 0。
; 获取第一个工作表 wsId = PbXls_GetSheetByIndex(wbId, 0) If wsId > 0 ; 可以操作工作表了 EndIf
PbXls_GetSheetTitle(wsId.i)
获取工作表的名称。
参数类型说明
wsIdInteger工作表ID
返回值： 工作表名称（字符串）。
PbXls_SetColumnWidth(wsId.i, col.i, width.f)
设置指定列的宽度。
参数类型说明
wsIdInteger工作表ID
colInteger列号（从1开始）
widthFloat列宽度值
返回值： 成功返回 #True，失败返回 #False。
; 设置第1列宽度为20 PbXls_SetColumnWidth(wsId, 1, 20.0) ; 设置第3列宽度为15.5 PbXls_SetColumnWidth(wsId, 3, 15.5)
PbXls_SetRowHeight(wsId.i, row.i, height.f)
设置指定行的高度。
参数类型说明
wsIdInteger工作表ID
rowInteger行号（从1开始）
heightFloat行高度值
返回值： 成功返回 #True，失败返回 #False。
PbXls_MergeCells(wsId.i, rangeString.s)
合并指定范围的单元格。范围格式为 "A1:B2" 或 "A1:D4"。
参数类型说明
wsIdInteger工作表ID
rangeStringString合并范围，如 "A1:B2"
返回值： 成功返回 #True，失败返回 #False。
; 合并A1到B2范围 PbXls_MergeCells(wsId, "A1:B2") ; 合并A1到D4范围 PbXls_MergeCells(wsId, "A1:D4")
PbXls_UnmergeCells(wsId.i, rangeString.s)
取消合并指定范围的单元格。
参数类型说明
wsIdInteger工作表ID
rangeStringString要取消合并的范围，如 "A1:B2"
返回值： 成功返回 #True，失败返回 #False。
PbXls_AppendRow(wsId.i, List values.s())
在工作表末尾追加一行数据。数据会从当前行的下一行开始，依次写入各列。
参数类型说明
wsIdInteger工作表ID
valuesList String要写入的字符串列表，每个元素对应一列
返回值： 成功返回 #True，失败返回 #False。
NewList data.s() AddElement(data()) data() = "姓名" AddElement(data()) data() = "年龄" AddElement(data()) data() = "城市" PbXls_AppendRow(wsId, data())

单元格操作

PbXls_SetCell(wsId.i, row.i, col.i, value.s)
设置指定单元格的值。单元格数据类型会根据值自动推断（字符串、数值、布尔等）。
参数类型说明
wsIdInteger工作表ID
rowInteger行号（从1开始）
colInteger列号（从1开始）
valueString单元格值
返回值： 成功返回 #True，失败返回 #False。
; 写入字符串 PbXls_SetCell(wsId, 1, 1, "Hello PbXls!") ; 写入数值 PbXls_SetCell(wsId, 1, 2, "123.45") ; 写入布尔值 PbXls_SetCell(wsId, 1, 3, "true") ; 写入日期 PbXls_SetCell(wsId, 1, 4, "2026-04-20")
PbXls_SetCellFormula(wsId.i, row.i, col.i, formula.s)
设置指定单元格的公式。公式必须以 "=" 开头。
参数类型说明
wsIdInteger工作表ID
rowInteger行号（从1开始）
colInteger列号（从1开始）
formulaString单元格公式，如 "=SUM(A1:A10)"
返回值： 成功返回 #True，失败返回 #False。
; 设置求和公式 PbXls_SetCellFormula(wsId, 11, 1, "=SUM(A1:A10)") ; 设置平均值公式 PbXls_SetCellFormula(wsId, 11, 2, "=AVERAGE(B1:B10)")

列操作

PbXls_InsertColumns(wsId.i, colIdx.i, count.i = 1)
在指定位置插入列。插入后，该列及右侧的所有单元格将向右移动，列宽和合并单元格范围会自动更新。
参数类型说明
wsIdInteger工作表ID
colIdxInteger插入位置的列号（从1开始）
countInteger要插入的列数，默认为1
返回值： 成功返回 #True，失败返回 #False。
; 在第3列前插入2列 PbXls_InsertColumns(wsId, 3, 2) ; 原来的C列及右侧所有列都向右移动2列
PbXls_DeleteColumns(wsId.i, colIdx.i, count.i = 1)
删除指定列。删除后，右侧的单元格将向左移动，列宽和合并单元格范围会自动更新。
参数类型说明
wsIdInteger工作表ID
colIdxInteger要删除的起始列号（从1开始）
countInteger要删除的列数，默认为1
返回值： 成功返回 #True，失败返回 #False。
; 删除第2列（B列） PbXls_DeleteColumns(wsId, 2, 1) ; 原来的C列及右侧所有列都向左移动1列

数据验证

PbXls_CreateDataValidation(type.s, sqref.s, formula1.s = "", formula2.s = "", operator.s = "between")
创建数据验证规则。支持多种验证类型：list（下拉列表）、whole（整数）、decimal（小数）、textLength（文本长度）、date（日期）、time（时间）、custom（自定义公式）。
参数类型说明
typeString验证类型，如 "list"、"whole"、"decimal" 等
sqrefString应用范围，如 "A1:A10"
formula1String公式或值（list类型用逗号分隔选项）
formula2String第二个公式（某些操作符需要）
operatorString运算符，如 "between"、"greaterThan" 等
返回值： 验证规则ID（整数）。
; 创建下拉列表验证 dv1 = PbXls_CreateDataValidation("list", "B2:B10", "选项1,选项2,选项3") ; 创建整数范围验证 dv2 = PbXls_CreateDataValidation("whole", "C2:C10", "18", "65", "between")
PbXls_SetValidationPrompt(validationId.i, title.s, message.s)
设置数据验证的输入提示信息。当单元格获得焦点时显示。
参数类型说明
validationIdInteger验证规则ID
titleString提示标题
messageString提示内容
返回值： 成功返回 #True，失败返回 #False。
PbXls_SetValidationError(validationId.i, title.s, message.s)
设置数据验证的错误提示信息。当输入无效值时显示。
参数类型说明
validationIdInteger验证规则ID
titleString错误标题
messageString错误内容
返回值： 成功返回 #True，失败返回 #False。

条件格式

PbXls_CreateConditionalFormat(type.s, sqref.s, formula1.s = "", formula2.s = "", operator.s = "greaterThan")
创建条件格式规则。支持类型：cellIs（单元格值比较）、expression（公式）、colorScale（颜色刻度）、dataBar（数据条）、iconSet（图标集）。
参数类型说明
typeString规则类型，如 "cellIs"、"expression"、"colorScale"
sqrefString应用范围，如 "A1:A10"
formula1String公式或比较值
formula2String第二个公式
operatorString运算符，如 "greaterThan"、"lessThan"、"between" 等
返回值： 规则ID（整数）。
; 单元格值大于85时红色粗体 cf1 = PbXls_CreateConditionalFormat("cellIs", "E2:E10", "85", "", "greaterThan") PbXls_SetConditionalFormatDxf(cf1, "FF0000", "", #True, #False) ; 颜色刻度（红-黄-绿） cf2 = PbXls_CreateConditionalFormat("colorScale", "D2:D10") PbXls_SetConditionalFormatColorScale(cf2, "FF0000", "FFFF00", "00FF00")
PbXls_SetConditionalFormatDxf(ruleId.i, fontColor.s = "", fillColor.s = "", fontBold.b = -1, fontItalic.b = -1)
设置条件格式的差异样式（Dxf），包括字体颜色、填充颜色、粗体和斜体。
参数类型说明
ruleIdInteger规则ID
fontColorString字体颜色，如 "FF0000"（红色）
fillColorString填充颜色，如 "FFFF00"（黄色）
fontBoldBoolean是否粗体（-1表示不变）
fontItalicBoolean是否斜体（-1表示不变）
返回值： 成功返回 #True，失败返回 #False。
PbXls_SetConditionalFormatColorScale(ruleId.i, minColor.s = "", midColor.s = "", maxColor.s = "", minType.s = "", midType.s = "", maxType.s = "")
设置颜色刻度的参数，包括最小/中间/最大颜色及其类型。
参数类型说明
ruleIdInteger规则ID
minColorString最小值颜色
midColorString中间值颜色
maxColorString最大值颜色
minTypeString最小值类型，如 "min"、"num"、"percentile"
midTypeString中间值类型
maxTypeString最大值类型，如 "max"、"num"、"percentile"
返回值： 成功返回 #True，失败返回 #False。

示例代码

示例1：创建基本Excel文件

; PbXls - 创建基本Excel文件示例 XIncludeFile "PbXls.pb" ; 创建工作簿 wbId = PbXls_CreateWorkbook() ; 获取第一个工作表 wsId = PbXls_GetSheetByIndex(wbId, 0) ; 写入表头 PbXls_SetCell(wsId, 1, 1, "姓名") PbXls_SetCell(wsId, 1, 2, "年龄") PbXls_SetCell(wsId, 1, 3, "城市") ; 写入数据行 PbXls_SetCell(wsId, 2, 1, "张三") PbXls_SetCell(wsId, 2, 2, "25") PbXls_SetCell(wsId, 2, 3, "北京") PbXls_SetCell(wsId, 3, 1, "李四") PbXls_SetCell(wsId, 3, 2, "30") PbXls_SetCell(wsId, 3, 3, "上海") ; 保存文件 PbXls_SaveWorkbook(wbId, "example1.xlsx") ; 清理资源 PbXls_Free()

示例2：创建带公式的Excel文件

; PbXls - 创建带公式的Excel文件示例 XIncludeFile "PbXls.pb" ; 创建工作簿 wbId = PbXls_CreateWorkbook() wsId = PbXls_GetSheetByIndex(wbId, 0) ; 写入表头 PbXls_SetCell(wsId, 1, 1, "项目") PbXls_SetCell(wsId, 1, 2, "金额") ; 写入数据 PbXls_SetCell(wsId, 2, 1, "工资") PbXls_SetCell(wsId, 2, 2, "10000") PbXls_SetCell(wsId, 3, 1, "奖金") PbXls_SetCell(wsId, 3, 2, "5000") PbXls_SetCell(wsId, 4, 1, "其他") PbXls_SetCell(wsId, 4, 2, "2000") ; 添加求和公式 PbXls_SetCell(wsId, 5, 1, "总计") PbXls_SetCellFormula(wsId, 5, 2, "=SUM(B2:B4)") ; 保存文件 PbXls_SaveWorkbook(wbId, "example2_formula.xlsx") ; 清理资源 PbXls_Free()

示例3：创建带样式的Excel文件

; PbXls - 创建带样式的Excel文件示例 XIncludeFile "PbXls.pb" ; 创建工作簿 wbId = PbXls_CreateWorkbook() wsId = PbXls_GetSheetByIndex(wbId, 0) ; 写入数据并设置列宽 PbXls_SetCell(wsId, 1, 1, "报表标题") PbXls_SetColumnWidth(wsId, 1, 30.0) ; 合并单元格 PbXls_MergeCells(wsId, "A1:C1") ; 写入多行数据 NewList row1.s() AddElement(row1()) row1() = "数据1" AddElement(row1()) row1() = "数据2" AddElement(row1()) row1() = "数据3" PbXls_AppendRow(wsId, row1()) ; 保存文件 PbXls_SaveWorkbook(wbId, "example3_style.xlsx") ; 清理资源 PbXls_Free()
